# Contador-Caracteres
![Badge versão do programa](https://img.shields.io/badge/version-1.0-9cf?style=for-the-badge "Version")

## Descrição
O projeto Contador-Caracteres consiste em um sistema para contar caracteres presentes e definindo seus tipos, utilizando a ferramenta spring boot para a criação da estrutura e desevolvido em linguagem Java(21).

## Funcionalidades
Lista de atualizações feitas durante o projeto:
- Fazer a leitura de numeros e caracteres na frase.
- Fazer a leitura de caracteres especiais na frase.
- Fazer a leitura de espaços contidos na frase.

## Exemplo de Interação do Usuario
Aqui está um exemplo de interação do usuário ao executar o programa:

```plaintext
Digite a frase (ou 'sair' para encerrar): teste pratico 123 !@#
A frase tem 12 letras, 3 dígitos, 3 caracteres especiais e 3 espaços
Digite a frase (ou 'sair' para encerrar): sair
Encerrando o programa.
```

Neste exemplo, o usuário escreve uma frase, recebe o resultado e encerra o programa após uma unica frase.
