# Contador-Caracteres
![Badge versão do programa](https://img.shields.io/badge/version-1.0-9cf?style=for-the-badge "Version")

Um projeto para contar caracteres presentes e definindo seus tipos

Lista de atualizações feitas durante o projeto:
- Fazer a leitura de numeros e caracteres na frase.
- Fazer a leitura de caracteres especiais na frase.
- Fazer a leitura de espaços contidos na frase.

Até o momento esse foi o projeto inicial, apenas para fins educativos.
